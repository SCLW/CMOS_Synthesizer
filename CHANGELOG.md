# Changelog

All notable changes to this project are documented here.

## [1.0.0] (2026-05-15)

First archived release of the CMOS Sound Experimentation Board documentation, deposited on Zenodo with a persistent DOI.

This release documents **Hardware Version 2.0 Revision B**, the production design completed in May 2024.
For the history of earlier revisions (Revision A, May 2024; 2020 prototype, since abandoned), see [README → Variants](README.md#variants).

### Included in v1.0.0

- PCB design files and schematics (Autodesk EAGLE)
- Bill of Materials for the main board and the LFSR deadlock-prevention add-on
- Assembly and mechanical drawings
- IC datasheets for offline reference
- Build guide
- Photo documentation from performances and workshops between 2024 and 2026

### Notes

- The schematic source files (`.brd`, `.sch`) carry nominal timing-capacitor values; the BOM specifies the recommended practical values. See [README → Appendix](README.md#appendix) for an explanation.
- Known issues are listed under [README → Errata](README.md#errata--known-issues).
